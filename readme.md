# README for TinyES3-Tools
The following tools are available for the daily use with the TinyES3:

   * es3discover
   * es3getpub
   * es3getpubsign
   * es3keygen
   * es3sign
   * es3slotlist
   * es3verify

More information are available here: https://www.emb4fun.de/projects/tes3/index.html

# Some notes about Mbed TLS
